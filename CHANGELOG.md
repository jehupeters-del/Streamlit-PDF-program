# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-13
### Added
- Streamlit-first PDF suite foundation with edit/merge, extraction, and validation workflows.
- Batch extraction ZIP export and batch validation CSV + TXT export.
- Copy-ready validation summary text area for clipboard workflows.
- Layered architecture (`app`, `src/domain`, `src/services`, `src/adapters`, `src/infrastructure`).
- CI quality gates for lint/format/type/test/build.
- Unit, integration, and e2e smoke tests with synthetic and real-world-style fixtures.
